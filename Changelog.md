# Changelog

All notable changes to this project will be documented in this file.

## [1.0.5] - 2026-07-23
### Added
- Added "Smart I/O Module" category supporting the A-80X (RS-485) and A-88X (Ethernet) series
  - A-80X: A-8012, A-8019, A-8051, A-8055/A-8055S, A-8057/A-8058, A-8060, A-8068/A-8069, A-8080
  - A-88X: A-8812, A-8819, A-8851, A-8855/A-8855S/A-8860, A-8869


## [1.0.4] - 2025-08-15
### Added
- Added support for 18xx and 19xx series I/O modules


## [1.0.3] - 2025-06-02
### Changed
- Updated the "Model" field to group available models by product category (e.g., AIoT, controller, ioModule) using `<optgroup>`, making it easier to select the appropriate model.
